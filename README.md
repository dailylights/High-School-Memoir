# 📓 High School Memoir · 青春纪念册

**一个轻量级、功能完整的高中回忆录分享网站，帮你珍藏与分享那段闪闪发光的高中时光。**

---

## 🌟 项目简介

你是否还记得同桌的纸条、教室的晚霞、操场上的奔跑？
「High School Memoir」是一个专为记录与分享高中生活而设计的开源网站项目。你可以在这里写下回忆、上传照片、参与话题、与老同学互动，共同构筑属于你们班级的青春档案馆。

> 🎯 **项目愿景**：让每个人的高中记忆，都有温度可循，有地方可回。

---

## ✨ 功能总览

### 🧑‍🤝‍🧑 用户系统
- 注册 / 登录 / 找回密码
- 个人资料编辑（昵称、头像、班级、手机号）
- 登录安全防护（失败次数限制、账号临时锁定）
- 会话超时自动登出

### 📝 回忆录
- 发布文字回忆录，支持关联话题分类
- 支持上传多张图片、视频、音频等多媒体
- 草稿箱：自动保存，随时续笔
- 回忆录标签系统（自定义标签、热门标签）
- 点赞、收藏、分享功能
- @提及功能（评论中 @ 用户，自动发送通知）

### 💬 互动社区
- 评论系统（支持二级回复、评论点赞、评论图片）
- 实时消息通知（点赞、评论、@提及通知）
- 关注 / 粉丝系统
- 私信聊天（支持文字 + 图片，实时对话）
- 用户黑名单（屏蔽不想看到的用户）

### 🏷️ 话题与搜索
- 话题广场：浏览所有话题，按话题筛选回忆录
- 全站搜索：支持回忆录、用户、相册、照片搜索
- 热搜词排行榜
- 搜索历史记录

### 📚 相册系统
- 创建个人相册，批量上传照片
- 照片点赞、相册点赞
- 相册封面自动生成
- 支持照片标题和描述

### 🏫 班级空间
- 创建 / 加入班级（邀请码加入）
- 班级专属回忆录墙
- 班级相册共享
- 班级成员管理
- 班级公告、班级活动、班级投票
- 班级权限体系（创建者 / 管理员 / 成员）

### 🎓 毕业纪念册
- 届别管理（按毕业年份组织）
- 毕业册时间轴展示
- 班级归属与届别关联

### 📢 公告系统
- 网站公告展示
- 管理员后台发布 / 编辑 / 删除公告

### 🔐 后台管理
- 📊 数据概览（用户数、回忆录数、评论数、班级数等统计）
- 👥 用户管理（查看、删除、设置/取消管理员）
- 📝 回忆录管理（查看、删除）
- 💬 评论管理（查看、删除）
- 🏷️ 话题管理（创建、删除）
- 📢 公告管理（发布、编辑、删除）
- 🏫 班级管理（创建、删除、自动生成邀请码）
- ⚙️ 系统配置（站点名称、描述、注册开关、安全策略等）
- 🔑 管理员账号设置（修改用户名、密码）

## 🛡️ 安全特性

| 安全防护 | 说明 |
|---------|------|
| **SQL注入防护** | 所有数据库查询使用预处理语句（Prepared Statements） |
| **XSS防护** | 所有前端输出均经过 escapeHtml 转义 |
| **CSRF防护** | 所有 POST 请求均经过 CSRF Token 验证 |
| **密码安全** | 使用 password_hash 加密存储，永不存明文 |
| **文件上传安全** | MIME类型检测 + 扩展名白名单 + getimagesize验证 + finfo验证 + 随机文件名 |
| **会话安全** | HttpOnly + SameSite + Secure Cookie，会话超时机制 |
| **登录防护** | 失败次数限制 + 临时锁定，防暴力破解 |
| **权限控制** | 后台操作严格校验管理员权限 |
| **安装锁定** | 安装完成后生成锁定文件，防止重复安装 |
| **安全响应头** | X-Content-Type-Options / X-Frame-Options / X-XSS-Protection / HSTS |

## 🚀 快速开始

### 1. 克隆项目

```bash
git clone https://github.com/dailylights/High-School-Memoir.git
cd High-School-Memoir
```

### 2. 环境要求

- PHP 7.0 或更高版本
- MySQL 5.7 或更高版本
- 启用 GD 扩展（图片处理）
- 启用 mysqli 扩展（数据库连接）

### 3. 安装向导

1. 将项目文件放入 Web 服务器目录
2. 确保 `uploads/` 目录和 `api/` 目录有写入权限
3. 在浏览器中访问 `install.html`
4. 按照向导步骤完成：
   - 环境检测
   - 数据库配置（支持连接测试）
   - 管理员账号设置
   - 系统配置
5. 安装完成后，系统会自动生成锁定文件

### 4. 启动服务（开发环境）

```bash
php -S localhost:8000
```

打开浏览器访问：**http://localhost:8000**

### 5. 重新安装

如需重新安装，请删除 `api/install.lock` 文件后再次访问 `install.html`。
